
#### 介绍
这是 SWJTU 计算机学院“从代码到实物：造你所想”课程的专属学习记录与项目源码仓库。
本仓库完整记录了我在整个课程周期中，从基础的电子元器件学习、电路板(PCB)绘制、ESP32 嵌入式开发、SolidWorks 3D结构建模，直到最终完成融合软硬云的 IoT 物联网产品原型的开发历程。

#### 仓库目录结构

本仓库主要包含以下目录与内容模块：

```text
my-project/
├── html/                # 🚀 课程全记录静态网页展板
├── final_project/         # 🌿 期末项目「智能植物卫士」软硬件资料
├── mid_term_project/      # ⏰ 中期项目「智能音乐时钟」3D打印外壳资料
├── esp32/                 # 🔌 ESP32 阶段各实验打样 Demo 代码
├── pcb/                   # 🖨️ 嘉立创 EDA 原理图与 PCB 绘图工程
├── 3D_design/             # 🧊 3D 打印外壳主源文件
├── SolidWorksLearning/    # ⚙️ SolidWorks 软件基础建模练习图纸
├── personal_introduction/ # 🙋 个人介绍图片
├── README.md              # 本说明文档

```

- **📁 weblog/**
  课程成果静态网页展板。使用前端技术详细记录了全程学习日志、中期项目和期末项目的图文与视频演示。
  👉 **快速查看**：在本地浏览器中直接打开 `weblog/index.html`，以获取最佳的图文全景可视化阅读体验。
  
- **📁 final_project/** 
  期末项目：**「智能植物卫士」**。包含基于 ESP32-C3 的完整物联网功能 C++ 源码、PlatformIO 编译配置、以及软硬件联调日志和 AI 修改建议表。
  
- **📁 mid_term_project/**
  中期项目：**「智能音乐时钟」**。包含项目的 3D 打印 3MF 格式切片文件，及装配调试工程资料等。
  
- **📁 esp32/**
  ESP32 阶段的实验打样代码（传感器模拟、WiFi通信、舵机控制等 Demo）。
  
- **📁 pcb/**
  嘉立创 EDA 课程学习与作业：原理图绘制、PCB 设计工程与布线走线实践。
  
- **📁 3D_design/ & SolidWorksLearning/**
  外形结构阶段学习：SolidWorks 建模练习图纸，以及各类外壳三维源文件。

#### 核心作品速览

1. **期末项目：智能植物卫士 (Smart Plant Guardian)**
   *一款可通过手机端远程监控并按需报警的室内智能植物看护终端。*
   使用了 ESP32 作为主控，外置多种环境感知传感器与彩色 TFT 显示屏，结合 WiFi 并打通平台。从分压电路查漏、代码开发到3D封装全部自主全栈完成。
   
2. **中期项目：智能音乐时钟**
   *外壳结构与打印工艺验证*
   核心关注于公差匹配、材料强度与内部硬件限位孔适配的 3D 打印装配验证项目。
